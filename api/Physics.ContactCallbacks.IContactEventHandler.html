Parameters

eventSource CollidableReference

Collidable that the event was attached to.

pair CollidablePair

Collidable pair triggering the event.

contactManifold TManifold

Set of remaining contacts in the collision.

contactOffset Vector3

Offset from the pair's local origin to the new contact.

contactNormal Vector3

Normal of the new contact.

depth float

Depth of the new contact.

featureId int

Feature id of the new contact.

contactIndex int

Index of the new contact in the contact manifold.

workerIndex int

Index of the worker thread that fired this event.

TManifold

Type of the contact manifold detected.
